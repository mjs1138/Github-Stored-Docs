How to Begin a GitHub project

  • Log into GitHub from a browser and create a Repo w/ a ReadMe file
  • Create a Repo w/ a ReadMe file
  • Using terminal window on users computer, create and/or change to the Parent folder that will be used for the project
  • Clone the project from command line Using the URL: https://GitHubUserName@github.com/GitHubUserName/RepositoryName.git (e.g git clone https://mjs1138@github.com/mjs1138/GitHub_HowTo.git)
  • Make Changes
  • Add/Commit changes as necessary
  • Push changes back to GitHub (should be doable w/ simple 'git push')
