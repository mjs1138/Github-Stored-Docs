• 'use'
    • Is for 'namespace' accessing
    • Does not use quotes (',")
    • Uses '\' separator
    • Still requires a 'include' or 'require'
  • 'include'
    • Is for including code from other files
    • Use quotes (',")
    • Uses '/' separator
  • 'set_include_path'
    • Sets a default path for includes
    • Use quotes (',")
    • Uses '/' separator
    • e.g. set_include_path("Level_1");
